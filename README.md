Fed's Bakery Website
A responsive bakery website built with HTML, CSS, and JavaScript.

About The Project
This website is for a fictional bakery called "Fed's Bakery". It showcases the bakery's products, allows customers to view the menu, place orders, and contact the bakery.

Features
Responsive design (mobile, tablet, desktop)

Online menu with filtering by category

Shopping cart for online orders (using localStorage)

Contact form for inquiries

Gallery of bakery items

How to Run
Download or clone the repository.

Open the index.html file in a web browser.

Alternatively, if you have a local server (like Live Server in VS Code), you can run it through that.

Project Structure
index.html - The main page (home page)

menu.html - The menu page with all the products

contact.html - The contact page with a form and location

style.css - All the styles for the website

script.js - All the JavaScript for interactive features

Technologies Used
HTML5

CSS3 (Flexbox, Grid, Media Queries)

JavaScript (ES6+)
